# Yogesh React App

This is a minimal Vite + React scaffold with Tailwind CSS, routing (`react-router-dom`) and icons (`react-icons`).

Prerequisites:
- Node.js (16+ recommended)
- npm (or yarn)

Install dependencies (run in project root):

```bash
npm install
# or install grouped packages explicitly:
# npm install react react-dom react-router-dom react-icons
# npm install -D vite @vitejs/plugin-react tailwindcss postcss autoprefixer
```

Initialize Tailwind (optional if you want presets):

```bash
npx tailwindcss init -p
```

Run dev server:

```bash
npm run dev
```

Build:

```bash
npm run build
npm run preview
```
